## /data
##### Description
The /data command will show you statistics that the bot has gathered while in use. This is, among others, the most searched items, command counters and uptime

##### Syntax
`/data`

##### Aliases
* `/d`
* `/stats`

##### Examples
`/data`
`/stats`
